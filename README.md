# Datasite Academy

Datasite Academy is a modern, responsive educational platform designed for data science enthusiasts. This application offers a seamless user experience for exploring courses, understanding the mission, and getting in touch with the academy.

## Features

- **Modern UI/UX**: Clean, responsive design built with React and custom CSS.
- **Navigation**: Intuitive routing using React Router.
- **Components**: Reusable and modular component architecture.

## Getting Started

### Prerequisites

Ensure you have Node.js installed on your system.

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Technologies

- React 19
- Vite
- React Router Dom
